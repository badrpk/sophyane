Sophyane Browser · AI browser shell

As flexible as modern AI browsers — open source on GitHub

Sophyane Browser gives you Ask / Search with sources, follow-ups, agent tools, mesh & hardware status, and history — a Perplexity-style flow for agent work. Download from the official GitHub repository. Opening the UI in a new tab of your current browser remains fully supported.

Download from GitHub

GitHub Releases browser/ folder on main Latest tarball (mirror)

Source of truth: github.com/badrpk/sophyane → browser/ and Releases assets.

1 · Install (Linux / macOS / ChromeOS)

curl -fsSL https://raw.githubusercontent.com/badrpk/sophyane/main/browser/install.sh | sh
sophyane-browser
install.sh Mirror tarball

2 · Windows (PowerShell)

irm https://raw.githubusercontent.com/badrpk/sophyane/main/browser/install.ps1 | iex
install.ps1

Or open without installing

Same home UI in a new tab of this browser — no download required.

Open Sophyane Browser UI in new tab →

CLI new-tab force: SOPHYANE_BROWSER_MODE=tab sophyane-browser

Features

Ask + sources

Perplexity-style query box, AI answers, source chips, follow-ups.

Agent tools

Platform, hardware, kernel, train status via local APIs.

Mesh & edge

Live mesh hello + hybrid/local answer modes.

Isolated profile

Chromium user-data dir under Sophyane state when available.

New-tab fallback

Always works in your existing browser if no Chromium.

Open source

Fully on GitHub — audit, fork, self-host.

After install

sophyane-browser
# dedicated Chromium window when Chrome/Chromium is installed

SOPHYANE_BROWSER_MODE=tab sophyane-browser
# force new tab in your default browser

# Optional: install Chromium for app-window mode
# Debian/Ubuntu: sudo apt install chromium
