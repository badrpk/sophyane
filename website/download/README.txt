Sophyane Browser — download package
===================================

Sophyane Browser is a Chromium-based shell with an isolated profile and
Sophyane home UI (mesh, hardware, chat, learn). If Chromium is not installed,
Sophyane still opens your normal browser in a NEW TAB — that behavior remains.

Linux / macOS / ChromeOS Linux / Termux
--------------------------------------
  curl -fsSL https://YOUR_HOST/download/install-sophyane-browser.sh | sh
  # or after download:
  bash install-sophyane-browser.sh

Windows (PowerShell)
--------------------
  irm https://YOUR_HOST/download/install-sophyane-browser.ps1 | iex

After install
-------------
  sophyane-browser
  sophyane --browser

Optional: install Chromium/Chrome for the dedicated app window.
