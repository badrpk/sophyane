1 · Create account / log in (2 minutes)

  1. Open Login / Get API key.
  2. Choose Sign up (new) or Log in (existing). Enter your email (and plan for signup).
  3. Click Send OTP to email. Check inbox / spam for “Sophyane verification code” from badrpk@gmail.com.
  4. Enter the 6-digit code → Verify OTP & get API key. Your key starts with sph_ — copy it once; it is not shown again in full.
  5. Test chat on the same page, or call the API (below).

Go to Login / Signup →

2 · Call the API

# Request OTP (signup once for new users)
curl -s https://HOST/api/v1/auth/request-otp \
  -H 'Content-Type: application/json' \
  -d '{"email":"you@mail.com","plan":"hybrid","purpose":"signup"}'

# Verify OTP → receive sph_ API key
curl -s https://HOST/api/v1/auth/verify-otp \
  -H 'Content-Type: application/json' \
  -d '{"email":"you@mail.com","otp":"123456","purpose":"signup"}'

# Chat
curl -s https://HOST/api/v1/chat \
  -H "Authorization: Bearer sph_YOUR_KEY" \
  -H 'Content-Type: application/json' \
  -d '{"message":"Hello Sophyane","edge":true}'

Replace HOST with this site’s domain (e.g. xerus.biz or your tunnel URL). Local dev: http://127.0.0.1:8780.
