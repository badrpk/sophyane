# Sophyane 19.0.0

Sophyane 19 introduces a dependency-free platform kernel for deep-agent software work.

## Included

- Repository indexing, lightweight symbol discovery, checkpoints, and rollback.
- Coded sandbox with workspace-only path resolution and capability manifests.
- Automatic snapshot deduplication and bounded log compaction.
- Deterministic project evaluation reports.
- Prompt guidance using Goal, Constraints, Context, Acceptance criteria, and Tests.
- Provider-neutral bounded sub-agents with roles, permissions, and local trace records.
- Local LangSmith-style JSONL run tracing.
- `sophyane-platform` command for status, indexing, checkpointing, evaluation, compaction, and prompt advice.

## Commands

```bash
sophyane-platform status
sophyane-platform index .
sophyane-platform checkpoint .
sophyane-platform eval .
sophyane-platform compact ~/.sophyane
sophyane-platform advise "build a mobile snake game"
```

The kernel does not require LangGraph, LangSmith, Docker, or an external database. It is designed to remain usable on Termux while allowing richer orchestration policies on larger devices.
